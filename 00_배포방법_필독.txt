[배포 순서]

1. 기존 GitHub/Vercel 프로젝트를 먼저 백업합니다.
2. 이 폴더의 파일을 기존 프로젝트에 덮어쓰되, 기존 .git 폴더와 .env 파일은 삭제하지 않습니다.
3. google-apps-script/Code.gs를 구글 Apps Script의 Code.gs에 전체 붙여넣고 새 버전으로 배포합니다.
4. 프로젝트 폴더에서 npm ci 후 npm run build가 성공하는지 확인합니다.
5. git add . / git commit / git push로 Vercel 재배포합니다.
6. Vercel 배포 완료 후 도메인 캐시를 새로고침합니다.
